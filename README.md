## Setup
Note: This implementation was taken from Thinking In React (https://reactjs.org/docs/thinking-in-react.html), Most of the code belongs to React, but has my personal twist :) 

1. Clone this repo
2. Run `npm install`
3. Run `npm start`
4. Happy Coding
